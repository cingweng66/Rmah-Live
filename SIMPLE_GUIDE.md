# 简单使用指南（小白版）

## 快速开始（3步）

### 第1步：启动服务

```bash
# 1. 启动后端（在 backend 目录）
cd backend
./start-docker.sh
# 或者如果没装 Docker：./start-local.sh

# 2. 启动前端（在项目根目录）
cd ..
npm install
npm run dev
```

### 第2步：注册账号（控制设备）

1. 打开浏览器访问：`http://localhost:8080`
2. 点击"注册"
3. 输入：
   - 姓名：你的名字
   - 邮箱：你的邮箱
   - 密码：你的密码
4. 点击"注册"按钮
5. **完成！** 系统会自动处理所有设置，直接进入控制面板

### 第3步：打开显示页面（显示设备）

在另一台设备（或同一台设备的另一个浏览器窗口）打开：

```
http://localhost:8080/#/display-public?room=default
```

或者选择其他布局：
- 布局2：`http://localhost:8080/#/display-public2?room=default`
- 布局3：`http://localhost:8080/#/display-public3?room=default`

**就这么简单！** 无需登录，无需配置，输入房间号就能显示。

## 使用流程

### 控制设备操作

1. **登录**：使用注册的邮箱和密码登录
2. **设置玩家**：在控制面板中设置玩家姓名、分数等
3. **开始游戏**：进行结算、立直等操作
4. **查看房间号**：在控制面板顶部可以看到房间号（默认是 `default`）

### 显示设备操作

1. **输入房间号**：在显示页面输入房间号（默认是 `default`）
2. **自动同步**：显示页面会自动接收控制设备的更新
3. **无需操作**：显示设备是只读的，会自动显示最新状态

## 常见问题

**Q: 注册后还需要做什么？**  
A: 不需要！注册后系统会自动创建 License，直接可以使用。

**Q: 登录后还需要激活 License 吗？**  
A: 不需要！如果没有 License，系统会自动创建一个。

**Q: 显示设备需要登录吗？**  
A: 不需要！显示设备只需输入房间号即可。

**Q: 房间号在哪里看？**  
A: 在控制面板的顶部，会显示当前房间号和显示设备访问地址。

**Q: 可以多个显示设备同时显示吗？**  
A: 可以！只要输入相同的房间号，多个设备都能同时显示。

## 提示

- 控制设备：需要登录，可以操作游戏状态
- 显示设备：无需登录，只需房间号，只能查看
- 房间号：默认是 `default`，可以在控制面板顶部看到
- 实时同步：所有操作都会实时同步到显示设备
